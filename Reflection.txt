Building this stock explorer showed how quickly MCP skill packs can turn a blank canvas into a production-ready app. The trickiest part was making the date-range slider actually meaningful: prices had to be re-indexed to 1.00 at the selected start date so the investment calculator returns period-specific growth rather than misleading all-time numbers. I was surprised how little code Streamlit needs — sidebar controls, interactive charts, and metric cards all fit under 100 lines. Playwright's screenshot automation was the most impressive moment: it navigated to the live URL, waited for the cold-start wake-up, and captured proof of deployment entirely hands-free. If I were to extend the app, I'd pull live prices from a real API so the data stays fresh beyond the built-in 2018-2019 dataset.
